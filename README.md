# python voice assistant
 Voice assitant in pyhton
 <h4>Features</h4>
 <ul>
 <li>It tell you the current time and date.</li>
 <li>It can tell you battery and cpu usage</li>
 <li>It can create the reminder list</li>
 <li>It can shut down or restart your system</li>
 <li>It can tell you non funny jokes</li>
 <li>It can open any website</li>
 <li>It can search things on wikipedia</li>
 <li>It can change my voice from male to female and vice-versa</li>
 </ul>
